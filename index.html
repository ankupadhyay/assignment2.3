1. html:Description: The root element of an HTML document that contains all other HTML elements
  2. head:Description: Contains metadata about the HTML document, such as the title, character encoding, and linked stylesheets.
  3. title: Description: Sets the title of the HTML document, which is displayed in the browser's title bar or tab.
  4. meta>:Description: Provides metadata about the HTML document, including character encoding, authorship, and viewport settings.
  5. link>:Description: Defines the relationship between the current document and an external resource, such as a stylesheet.
  6. style>: Description: Used to embed CSS (Cascading Style Sheets) within an HTML document to define its visual appearance.
  7. script>:Description: Embeds or references JavaScript code, allowing dynamic behavior and interactivity on the webpage.
  8. body>:Description: Contains the visible content of the HTML document, including text, images, links, and other elements.
  9. header>:Description: Typically used to represent introductory content at the top of a webpage, such as a logo, navigation, or site title.
  10. nav>:Description: Defines a section of navigation links, often found within the header or footer of a webpage.
  11. main>: Description: Contains the primary content of the webpage, excluding header, footer, and sidebars.
  12. article>: Description: Represents a self-contained piece of content, such as a blog post, news article, or forum post.
  13. section>:Description: Groups related content within a webpage, often used for structuring layout or thematic organization.
  14. aside>:Description: Typically used for content that is tangentially related to the main content, such as sidebars, advertisements, or additional information.
  15. footer>:Description: Represents the footer section of a webpage, often containing copyright information, contact details, or links to related pages.
  16. h1, h2, h3, h4, h5, h6:Description: Headings that define the hierarchical structure of the document, with h1> being the highest level and h6> the lowest.
  17. p:Description: Defines a paragraph of text, separating it from other content on the page.
  18. a:Description: Creates hyperlinks to other web pages, documents, or resources by specifying the target URL.
  19. img:Description: Embeds an image in the document, allowing for the display of graphics and illustrations.
  20. ul and ol: Description: ul> creates an unordered (bulleted) list, while ol> creates an ordered (numbered)list. Both are used to list items.
  21. Description: Defines a list item within ul> or ol> elements.
